string AcUtils.Version.ToString ( string  format,
IFormatProvider  provider 
)
inline

The ToString implementation.

Parameters
formatThe format specifier to use, e.g. Console.WriteLine(ver.ToString("r"));
providerAllow clients to format output for their own types using ICustomFormatter.
Returns
The formatted string.
Exceptions
FormatExceptionthrown if an invalid format string is specified.
Format specifiers:
  • LV Long version (verbose).
  • G Location, the depot-relative path of the element (default when not using a format specifier).
  • I Element ID.
  • T The element's type: dir, text, binary, ptext, elink, or slink.
  • F True if the element is a folder, False otherwise.
  • V Virtual stream\version number format, e.g. MARS_STAGE\17 (2\17)
  • R Real stream\version number format, e.g. MARS_DEV3_barnyrd\8 (52\8)
  • A Ancestor stream\version number format.
  • M Merged-in stream\version number format.
  • MN Initialized using first part of mergedAgainstNamedVersion XML attribute, e.g. mergedAgainstNamedVersion="MARS_DEV3_barnyrd/8"
  • AN Initialized using first part of ancestorNamedVersion XML attribute.
  • MG Time the merge took place.
  • CS Checksum of the version.
  • FS File size in bytes.
